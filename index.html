Chestnut by astromme

Parallel Programming For Lazy People

Consisting of a language and a graphical environment, Chestnut is an easy-to-learn, easy-to-use graphics card programming tool. Chestnut evolved out of Andrew Stromme Ryan Carlson's final project for CS87 - Parallel and Distributed Computing at Swarthmore College.

Example

int2d unsorted[10, 5] = :random(80, 100);
int2d sorted[10, 5] = :sort(unsorted);

int reduced = :reduce(sorted);

print(unsorted);
print(sorted);
print("Reduced = %s", reduced);
